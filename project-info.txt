                                QUICK START
================================================================================

1. Extract this archive
2. cd fairsplit-ai
3. npm install
4. cp .env.example .env
5. Add your OpenAI API key to .env
6. npx prisma db push
7. npm run db:seed
8. npm run dev
9. Visit http://localhost:3000

Default login: alice@example.com / password123
